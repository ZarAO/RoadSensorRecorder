Contributing
============

Thanks for your interest in contributing to RoadSensorRecorder.

How to contribute
-----------------
- Fork the repository.
- Create a feature branch: `git checkout -b feat/short-description`.
- Make your changes and run the build locally.
- Commit your changes with clear messages.
- Open a pull request and describe the change.

Code style
----------
- Java code should follow standard Android/Java conventions.
- Keep commits small and focused.

Reporting bugs
--------------
- Open an issue with steps to reproduce and relevant logcat output.
